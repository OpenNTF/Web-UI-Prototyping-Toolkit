Create a new Portal theme maven project tree

This is a tool for theme developers - it sets up a maven project tree that will produce a fresh EAR upon 'mvn package'

The theme is set up to host dynamic and static files as WAR packaged inside of an EAR.

Custom renamed dynamic content spots are defined in the plugin.xml inside the dynamic files WAR.

All references should be modified to load from WAR rather than WebDAV.
